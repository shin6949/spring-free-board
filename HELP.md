## Free Board
Spring Boot로 만든 간단한 자유 게시판. 아래 기능을 포함할려고 합니다.

- 회원가입
  - 이름
  - 아이디 (중복 확인 필수)
  - 닉네임 (중복 확인 필수)
  - 비밀번호
  - E-mail (중복 확인 필수)
- 로그인
- 마이 페이지
  - 작성한 게시글
  - 개인 정보 수정
  - 프로필 수정
  - 프로필 사진 지정
- 글 작성
  - 네이버 스마트 에디터 적용
  - 사진 업로드
- 글 조회 (뷰어)
  - 페이지 기능
  - 조회수 카운팅
  - 최신 순 보기
  - 추천 순 보기 (1계정+1게시글=1추천)
  - 조회수 순 보기
- 글 검색
  - 제목만
  - 제목 + 내용
  - 작성자
  - 댓글 내용
  - 댓글 작성자
- 댓글
  - 댓글 작성
  - 댓글 수정
  - 댓글 삭제
  - 댓글 추천 (1계정+1댓글=1추천)
- 답글
  - 위 댓글과 기능 동일
- 추천 기능

## Tech Stack

- JDK: 11
- Spring Boot 2.7.6
- DB: MySQL / mariaDB
- Persistent: MyBatis
- Spring Security
